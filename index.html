MultitouchPadOsc

an application to send multitouch messages via osc
Introduction | Documentation | Download | Contact

Send Apple Trackpad or
MagicTrackpad data via osc

This application was developed to connect several Magic-Trackpads to a Master Computer.


Learn more about MultitouchPadOsc at the documentation. Information about different scenarios can be found here.
